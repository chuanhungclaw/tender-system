# GitHub Pages 設定

## 部署步驟
1. 訪問：https://github.com/chuanhungclaw/tender-system/settings/pages
2. 設定：
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (根目錄)
3. 點擊 Save
4. 等待 2 分鐘
5. 訪問：https://chuanhungclaw.github.io/tender-system/

## 檔案結構
- `index.html` - 主頁面
- `style.css` - 樣式表
- `app.js` - JavaScript 邏輯
- `.nojekyll` - 禁用 Jekyll

## 技術棧
- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage
- GitHub Pages
