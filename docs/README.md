## 🎯 業務規則

### 標案評分系統

```
綜合評分 = 100 分滿分

1. 利潤潛力（30 分）
   - 預算金額
   - 預估利潤率
   - 付款條件

2. 得標機率（25 分）
   - 歷史得標記錄
   - 競爭激烈程度
   - 規格匹配度

3. 履約風險（20 分）
   - 機關付款記錄
   - 產品複雜度
   - 交貨時間壓力

4. 戰略價值（15 分）
   - 是否為重複採購機關
   - 是否能建立案例
   - 行業影響力

5. 資源匹配（10 分）
   - 現有供應鏈配合度
   - 設計能力匹配
   - 產能負荷
```

### 推送規則

- **時間：** 每天早上 8:00
- **頻道：** Telegram
- **頻率：** 每天
- **內容：** 新標案清單 + AI 分析 + 建議行動
