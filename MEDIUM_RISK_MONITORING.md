## 🚨 異常檢測規則

### 觸發警報的條件

**高頻率操作：**
- 定時任務 > 10 次/小時
- 日程操作 > 20 次/小時
- 待辦操作 > 50 次/小時

**異常時間：**
- 凌晨 2:00-6:00 執行任務
- 假日大規模操作

**敏感操作：**
- 批量刪除
- 訪問未授權數據
- 修改他人數據

## 📞 應急處理

### 發現異常時

1. **立即停止**相關 Skill
2. **記錄**異常操作詳情
3. **通知**用戶
4. **調查**原因
5. **修復**並防止再次發生

### 禁用 Skill 命令

```bash
# 禁用特定 Skill
cd ~/.openclaw/extensions/wecom/skills
mv wecom-schedule wecom-schedule.DISABLED

# 恢復 Skill
mv wecom-schedule.DISABLED wecom-schedule
```
