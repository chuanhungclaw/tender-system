# GitHub Pages 啟用指南

## 步驟 1: 訪問 GitHub Pages 設定

**URL:** https://github.com/chuanhungclaw/tender-system/settings/pages

## 步驟 2: 設定 Source

1. **Source**: 選擇 **Deploy from a branch**
2. **Branch**: 選擇 **main**（或 gh-pages-deploy）
3. **Folder**: 選擇 **/**（根目錄）

## 步驟 3: 儲存設定

點擊 **Save** 按鈕

## 步驟 4: 等待部署

等待 **2-3 分鐘**，GitHub Pages 會自動部署

## 步驟 5: 訪問網站

**網站網址:** https://chuanhungclaw.github.io/tender-system/

## 檔案清單

- `.nojekyll` - 禁用 Jekyll 處理
- `CNAME` - 自訂網域設定（可選）
- `index.html` - 主頁面
- `style.css` - 樣式表
- `app.js` - JavaScript 邏輯

## 技術規格

- **平台**: GitHub Pages
- **部署時間**: 2-3 分鐘
- **SSL**: 自動啟用
- **網域**: chuanhungclaw.github.io/tender-system/
- **狀態**: 等待啟用
